## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/ABflames/Python-basic-to-advanced.git
```

2. Navigate to the project folder

```bash
cd Python-basic-to-advanced/text_rpg
```

3. Run the game

```bash
python3 rpg.py
```
